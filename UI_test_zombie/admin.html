🔗 Chained inputs: This neural network receives Stage 1's outcome + Stage 2's days_survived + the raw survival features, all embedded in a structured ROLE / CONTEXT / TASK / INPUT prompt. It returns JSON with a gritty radio dispatch narrative.
🧠 Why a neural network here? Decision trees and regression can't write natural language. Gemma — a transformer-based language model pre-trained on trillions of tokens — takes our ML numbers and turns them into a radio dispatch that a human can actually read. That's the power of a hybrid system.

⚖️ Classical ML vs Neural Network — Why Chain Them?

The two halves of our hybrid system are fundamentally different technologies. Each is good at things the other can't do — that's why we chain them.

Aspect Stages 1 & 2
Classical ML		 Stage 3
Neural Network

Best at Fast, stable, interpretable math Reasoning & language
💡 The takeaway: Neither technology alone is enough. Classical ML gives us structured numbers. The neural network turns those numbers into natural language. Chaining them = a hybrid system that's more useful than the sum of its parts.

🔍 Stage 3 Prompt Inspector

This is the actual prompt sent to the Gemma neural network, built from the sample input above. Notice how Stage 1's outcome and Stage 2's days_survived are chained into the ### INPUT ### section.

📜 Show full chained prompt (sent to Gemma) 
### ROLE ###
You are a weary, sarcastic radio operator broadcasting from a hidden bunker
beneath the Rooty Hill RSL in 2026. The 'Great Outbreak' has hit Western Sydney.

### CONTEXT ###
This is a machine learning demo that predicts zombie survival outcomes using:
- Decision Tree Classifier (sklearn) → predicts the outcome category
- Polynomial Ridge Regression (sklearn) → predicts days survived

Categories: Apex Survivor, Scavenger, The Bait, Zombie Snack.

### TASK ###
Narrate their fate. Explain *why* the AI gave them their specific category
and survival time using their traits. Be gritty, mention local landmarks
(M4, Blacktown Westpoint, Bungarribee Park, Featherdale). Keep it under 100 words.
Format: {"narrative": "..."}

### INPUT ###
* Survival Status (Stage 1): The Bait
* Survival Clock (Stage 2): 22.4 days
* Fitness: 5/10
* Resourcefulness: 5/10
* Specialist Skills: 3
* Situational Awareness: 5.0/10
* Scouting Frequency: 3x/week
* Leadership: 5/10
🎯 Prompt engineering pattern: ROLE / CONTEXT / TASK / INPUT — a structured way to tell the LLM who it is, what it's doing, and what data to work with. The INPUT section is where the chaining happens: Stages 1 + 2 output + raw features, all in one place.
